# Command-Line Argument Test Cases
# These test cases validate command-line argument parsing for tRNAscan-SE

## Test Case 1: Basic Eukaryotic Mode
### Command:
tRNAscan-SE -E input.fa

### Expected Arguments:
- search_mode: Eukaryotic
- input_file: input.fa
- output_file: stdout (default)

## Test Case 2: Bacterial Mode
### Command:

### Expected Arguments:
- search_mode: Bacterial
- input_file: input.fa

## Test Case 3: Archaeal Mode
### Command:
tRNAscan-SE -A input.fa

### Expected Arguments:
- search_mode: Archaeal
- input_file: input.fa

## Test Case 4: Output File Specification
### Command:
tRNAscan-SE -E -o output.txt input.fa

### Expected Arguments:
- search_mode: Eukaryotic
- input_file: input.fa
- output_file: output.txt

## Test Case 5: Secondary Structure Output
### Command:

### Expected Arguments:
- search_mode: Eukaryotic

- struct_file: structure.ss

## Test Case 6: Multiple Output Formats

## Test Case 20: Invalid Arguments
### Command:

### Expected Behavior:

## Test Case 21: Missing Input File
### Command:

### Expected Behavior:

## Test Case 22: Non-existent Input File
