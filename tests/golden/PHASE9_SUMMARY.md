## Critical Output Fields Documented

### Tabular Output (.out)
| Field | Format | Validation Rule |
|-------|--------|----------------|
| Sequence Name | String | Exact match from input |
| tRNA # | Integer | Sequential per sequence |
| Begin | Integer | 1-indexed, Begin > End for reverse |
| End | Integer | 1-indexed |
| Type | 3-char | Valid AA code or SeC |
| Codon | 3-char | Uppercase nucleotides |
| Intron Begin/End | Integer | 0 if none |
| Inf Score | Float | `%.1f` format |
| HMM Score | Float | `%.2f` format |
| 2'Str Score | Float | `%.2f` format |
| Hit Origin | String | Inf, Ts, Eu, Bo |
| Isotype CM | 3-char | Best isotype model |
| Isotype Score | Float | `%.1f` format |
| Note | String | Optional (ISM, pseudo) |

## Command Lines Used (Original)

### Example1 (Eukaryotic Mode)
```bash
tRNAscan-SE -E -H \
  -o Example1-tRNAs.out \
  -f Example1-tRNAs.ss \
  -m Example1-tRNAs.stats \
  -b Example1-tRNAs.bed \
  -s Example1-tRNAs.iso \
  Example1.fa
```

### Example2 (Eukaryotic Mode)
```bash
tRNAscan-SE -E -H \
  -o Example2-tRNAs.out \
  -f Example2-tRNAs.ss \
  -m Example2-tRNAs.stats \
  -s Example2-tRNAs.iso \
  Example2.fa
```

**Options:**
- `-E` - Eukaryotic mode
- `-H` - Show score breakdown (HMM + 2'Str)
- `-o` - Tabular output file
- `-f` - Secondary structure file
- `-m` - Statistics file
- `-b` - BED format file
- `-s` - Isotype-specific scores file
