# Covariance Model Information for Bacterial Isotype Scoring

## Model Database Files

### TRNAinf-bact-iso (Main Database)
- **Location**: `original/lib/models/TRNAinf-bact-iso`
- **Format**: Concatenated Infernal 1.1 CM database
- **Size**: ~1.2 MB
- **Number of models**: 23 isotype-specific models
- **Version**: Infernal 1.1.1 (July 2014)
- **Calibrated**: Yes (with E-values and bit score thresholds)

### Separate SeC Model
- **File**: `original/lib/models/TRNAinf-bact-SeC.cm`
- **Purpose**: Selenocysteine detection (21st amino acid)
- **Special**: Recognizes TCA anticodon that suppresses UGA stop codon

## Scoring Algorithm

### cmsearch Command Line
```bash
cmsearch --tblout output.tbl \
         --noali \
         --cpu 1 \
         TRNAinf-bact-iso \
         candidate.fa
```

### Output Format (--tblout)
```
# target name       accession  query name           accession  mdl mdl from   mdl to seq from   seq to strand trunc pass   gc  bias  score   E-value inc description
CELF22B7.trna1      -          bact-Leu             -           cm        1       76        1       76      +    no    1 0.49   0.0  119.9   3.5e-23 !   -
```

### Bit Score Calculation
- **Raw score**: From dynamic programming algorithm
- **Null model**: Random sequence model for baseline
- **Bit score**: log₂(P(seq|model) / P(seq|null))
- **E-value**: Expected number of hits with this score or better by chance

### Score Ranges (Typical)
- **Excellent match**: > 100 bits
- **Good match**: 60-100 bits
- **Marginal match**: 30-60 bits
- **Poor match**: < 30 bits
- **Structural mismatch**: -999 (cannot score)

## Model-Specific Features

### Type I tRNAs (Standard)
- **Structure**: 4 bp variable loop
- **Length**: ~76 bp
- **Models**: Most isotypes (Ala, Arg, Asn, Asp, etc.)

### Type II tRNAs (Long Variable Arm)
- **Structure**: 13-21 bp variable loop
- **Length**: ~85-95 bp
- **Models**: Ser, Leu only
- **Special**: Extra stem-loop in variable region

### SeC (Selenocysteine) tRNAs
- **Structure**: 8 bp acceptor stem (vs. 7 bp standard)
- **Structure**: 6 bp D-stem (vs. 4 bp standard)
- **Length**: ~95 bp
- **Anticodon**: TCA (recognizes UGA stop codon)
- **SECIS**: Requires SECIS element in mRNA for insertion

### iMet/fMet (Initiator Methionine)
- **Structure**: 3 mismatches in acceptor stem (1:72, 2:71, 3:70)
- **Purpose**: Translation initiation
- **Bacterial**: Often N-formylated (fMet)
- **Different**: From elongator Met-tRNA
