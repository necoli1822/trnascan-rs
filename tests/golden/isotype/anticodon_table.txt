# Standard Genetic Code: Anticodon to Isotype Mapping
# Format: Anticodon	Isotype	Codon(s)	Notes

# This table shows the expected isotype for each anticodon based on the standard genetic code
# Anticodons are written 3'->5' (as they appear in the tRNA sequence)
# The actual codon they recognize is the reverse complement

# Alanine (Ala)
AGC	Ala	GCU,GCC,GCA,GCG
CGC	Ala	GCG
TGC	Ala	GCA
GGC	Ala	GCC

# Arginine (Arg)
ACG	Arg	CGU,CGC
CCG	Arg	CGG
TCG	Arg	CGA
GCG	Arg	CGC
CCT	Arg	AGG
TCT	Arg	AGA	Wobble

# Asparagine (Asn)
GTT	Asn	AAC,AAU

# Aspartate (Asp)
GTC	Asp	GAC,GAU

# Cysteine (Cys)
GCA	Cys	UGC,UGU

# Glutamine (Gln)
CTG	Gln	CAG
TTG	Gln	CAA

# Glutamate (Glu)
CTC	Glu	GAG
TTC	Glu	GAA

# Glycine (Gly)
ACC	Gly	GGU,GGC
GCC	Gly	GGC
TCC	Gly	GGA
CCC	Gly	GGG

# Histidine (His)
GTG	His	CAC,CAU

# Isoleucine (Ile)
AAT	Ile	AUU
GAT	Ile	AUC
TAT	Ile	AUA

# Leucine (Leu)
CAA	Leu	UUG
TAA	Leu	UUA
CAG	Leu	CUG
TAG	Leu	CUA
AAG	Leu	CUU	Wobble
GAG	Leu	CUC

# Lysine (Lys)
CTT	Lys	AAG
TTT	Lys	AAA

# Methionine (Met)
CAT	Met	AUG	Start codon

# Initiator Methionine (iMet)
CAT	iMet	AUG	Initiator form, different structure

# Formyl-Methionine (fMet)
CAT	fMet	AUG	Bacterial initiator, N-formylated

# Phenylalanine (Phe)
GAA	Phe	UUC,UUU

# Proline (Pro)
AGG	Pro	CCU,CCC
CGG	Pro	CCG
TGG	Pro	CCA
GGG	Pro	CCC

# Selenocysteine (SeC / Sec)
TCA	SeC	UGA	Suppresses UGA stop codon; 21st amino acid

# Serine (Ser)
ACT	Ser	AGU,AGC	Has variable arm
GCT	Ser	AGC
TGA	Ser	UCA	Has variable arm
CGA	Ser	UCG	Has variable arm
AGA	Ser	UCU
GGA	Ser	UCC

# Threonine (Thr)
AGT	Thr	ACU,ACC
CGT	Thr	ACG
TGT	Thr	ACA
GGT	Thr	ACC

# Tryptophan (Trp)
CCA	Trp	UGG

# Tyrosine (Tyr)
GTA	Tyr	UAC,UAU

# Valine (Val)
AAC	Val	GUU
CAC	Val	GUG
TAC	Val	GUA
GAC	Val	GUC

# Stop Codons (Suppressor tRNAs)
TTA	Sup	UAA	Ochre suppressor
CTA	Sup	UAG	Amber suppressor
TCA	Sup	UGA	Opal suppressor (or SeC)

# Notes:
# 1. Wobble pairing allows one anticodon to recognize multiple codons
# 2. SeC (Selenocysteine) is encoded by a UGA stop codon with special machinery
# 3. Some organisms have additional anticodons or modified genetic codes
# 4. Ser and Leu isotypes have long variable arms (Type II tRNAs)
# 5. Bacterial tRNAs may have post-transcriptional modifications affecting anticodon
